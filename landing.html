Features

Built for real analysis

Not a toy demo. A full-stack application with file parsing, persistent chat history, and real-time AI streaming.

↑

Smart File Upload

Drop CSV, Excel, PDF, or text files. Server-side parsing extracts structured data for AI context.

›_

Real-time Streaming

Responses stream word-by-word via SSE. No waiting for complete responses. Instant feedback.

#

Rich Markdown

Code blocks with copy button. Tables, links, bold, italic, headings. Beautifully formatted output.

□

Chat History

SQLite-backed persistence. Create, rename, delete conversations. Pick up where you left off.

◑

Dark & Light Mode

Full semantic design system. Toggle between themes with every element adapting instantly.

</>

Embeddable Widget

How it Works

Three simple steps

Clone, run, analyze. No config files. No environment variables. One command to start.

01

Clone & Setup

Clone the repo and run the setup script. It installs dependencies, checks TypeScript, and starts both servers automatically.

02

Upload Data

Drag a CSV, Excel spreadsheet, or PDF into the chat. The backend parses it and extracts structured data as context.

03

Ask Questions

Ask anything about your data. Get streaming responses with formatted tables, insights, and recommendations.
